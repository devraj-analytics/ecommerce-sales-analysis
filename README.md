# E-commerce Sales & Customer Analysis

Project Overview
This project analyzes e-commerce sales data using SQL and Power BI to uncover insights about revenue trends, customer behavior, and payment preferences.

Tools Used
- SQL Server
- Power BI

Key Analysis
- Total Revenue and Order Volume
- Monthly Sales Trends
- Top 10 Customers by Revenue
- Top Product Categories
- Payment Method Analysis
- Average Order Value (AOV)

Key Insights
- Revenue peaked in November 2017, indicating seasonal demand
- Credit cards contribute the majority of revenue (~78%)
- A small group of customers contributes a significant portion of revenue
- Product demand is concentrated in a few categories
- Average order value suggests potential for upselling
